Política de privacidad

Secciones
1. Información que recopilamos ▾

ListUp no solicita información personal como nombre, teléfono o dirección. La app guarda localmente información que tú introduces: nombres de listas, productos y su estado (comprado o no).

2. Cómo usamos la información ▾

Usamos la información únicamente para permitirte usar la app: crear listas, guardar productos y gestionar compras. No usamos tus datos para publicidad personalizada.

3. Almacenamiento y sincronización ▾

Tus datos se almacenan en el dispositivo usando una base de datos local (SQLite). Por defecto, ListUp no sincroniza datos con servidores. Si en el futuro se agrega sincronización o backup, se informará claramente y se actualizará esta política.

4. Compartición con terceros ▾

No vendemos ni compartimos tus datos con terceros. No transferimos tus listas a servicios externos.

5. Permisos ▾

Actualmente ListUp no requiere permisos sensibles para funcionar. Si en el futuro se requieren permisos (por ejemplo, notificaciones o almacenamiento para exportación), se solicitarán con transparencia y podrás decidir.

6. Seguridad ▾

Tus datos permanecen en tu dispositivo. Recomendamos usar bloqueo de pantalla (PIN/biometría). Ningún método es 100% infalible, pero tomamos buenas prácticas para reducir riesgos.

7. Privacidad de menores ▾

ListUp no está dirigida específicamente a menores y no recopila datos personales de menores. Si crees que un menor nos ha enviado información, contáctanos para revisarlo.

8. Cambios a esta política ▾

Podemos actualizar esta política cuando sea necesario. Publicaremos cambios en esta misma página y actualizaremos la fecha de “Última actualización”.
